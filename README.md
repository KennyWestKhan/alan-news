# news

## Project setup

```
get Alan SDK Key from (https://studio.alan.app/embed/news-app)
get news api key from (https://newsapi.org/account)
```

```
add to environment variables VUE_APP_ALAN_KEY and VUE_APP_NEWS_KEY
```

```
npm install
```
